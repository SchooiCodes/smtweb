About



SMT (a.k.a. Schooi's Multitool) is a tool created by me that aims to provide several other tools that can help users (mainly developers) in their daily life.


It started as a small project I came up with to store my batch projects, but it really quickly ended up as a very powerful multitool.
I have never seen a multitool that was created with Batch (the programming language) this big.

Features

SMT has over 70 TOOLS!


Some important ones are:

  • IP Geolocator
  • SMB Bruteforcer
  • Various App Installers
  • CLI Task Manager
  • Network Speedtest
  • Text to Speech
  • Website Blocker
  • .zip File Password Cracker
  • Windows User Manager
  • Password Generator
