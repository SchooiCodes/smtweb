About

SMT (Schooi's Multitool) is a tool created by me that aims to provide several tools to help users, mainly developers, in their daily tasks.

It started as a small project to store my batch projects but quickly evolved into a powerful multitool. I have never seen a multitool of this scale created with Batch.

Features

SMT has over 70 TOOLS!

Some important ones are:

  • IP Geolocator
  • SMB Bruteforcer
  • Various App Installers
  • CLI Task Manager
  • Network Speedtest
  • Text to Speech
  • Website Blocker
  • .zip File Password Cracker
  • Windows User Manager
  • Password Generator

Downloads
